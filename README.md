# Web for Medical Record

*TAG: #C# #ASP.NET_Core #MVC #API #Vue #axios #SQL*

連接資料庫，提供多種查詢方式以便搜尋病歷。可從頁面編輯、刪除資料，並將更新送回資料庫。
- ASP.NET Core Web (MVC) + Web API 架構
- 連接 SQL 資料庫
- 使用 Vue.js、 moment.js、 jQuery UI 渲染前端
- 使用 axios 與後端Web API串接
